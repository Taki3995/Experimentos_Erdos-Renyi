The Random Network Model G(N,p) 1
A random network consists of N nodes where each node pair is connected
with probability p.
The steps to generate a random network are the following:
1 Start with N isolated nodes.
2 Select a node pair and generates a random number between 0 and 1.
If the number exceeds p connect the nodes with a link, otherwise
leave them disconnected.
3 Repeat step 2 for each of the N(N −1)/2 node pairs.

Small Worlds
The small-world phenomenon a.k.a. six degrees of separation states
that between any two individuals exists a path of at most six
acquaintances.
In terms of network science language this phenomenon implies that
the distance between two randomly chosen nodes in a network is
short.
